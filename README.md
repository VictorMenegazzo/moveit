# How to use

To start your Next.js website:

```
# Clone this repository
$ https://github.com/VictorMenegazzo/moveit.git

# Install dependencies
$ yarn install

# Start the project
$ yarn dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
